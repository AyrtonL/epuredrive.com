Key insight: one repo, three surfaces

Next.js middleware inspects the hostname on every request before routing. A request to ayrtonn.epuredrive.com gets rewritten to the tenant fleet route with slug = "ayrtonn". A request to epuredrive.com goes to the marketing site. A request to epuredrive.com/dashboard is auth-guarded and shows the admin app. All from one codebase, one deployment.
