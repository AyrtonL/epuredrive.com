Route groups — the key trick

The (marketing) and (tenant) folders use Next.js route groups — the parentheses mean they don't affect the URL path, but they let each surface have its own layout, loading state, and metadata. The middleware reads the hostname first and rewrites the request to the correct group before the router sees it. Same page at /, completely different UI depending on which domain you're on.

Migration note — Netlify functions stay

Your existing Netlify functions (Stripe, iCal sync, Gmail, etc.) don't need to be rewritten. They stay in netlify/functions/ and keep working. We migrate the frontend first, functions later only if needed.
