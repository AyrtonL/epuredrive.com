PRIORITY 1: REQUIRED BEFORE GOING LIVE
---------------------------------------

1. STRIPE — Create new Pro price at $149/mo
   - Go to: https://dashboard.stripe.com/prices
   - Create new recurring price: $149/month
   - Copy the new price_id (starts with price_...)
   - Update these files with the new price_id:
     • admin/js/dashboard.js → PRO_PRICE_ID constant (line ~46)
     • netlify/functions/stripe-webhook.js → PRO_PRICE_ID constant (line 11)
   - Verify Starter price is $49/mo (current: price_1TDaQ3HAH4zJnnwfasGBYtYO)
   - Archive the old $99/mo Pro price in Stripe

2. SUPABASE STORAGE — Create tenant-logos bucket
   - Go to: https://supabase.com/dashboard → Storage
   - Create new bucket: "tenant-logos"
   - Make it PUBLIC (so logos can be displayed on fleet pages)
   - Allow file types: image/png, image/jpeg, image/svg+xml, image/webp

3. DNS — Set up wildcard subdomain
   - Go to your DNS provider for epuredrive.com
   - Add record: *.epuredrive.com CNAME → epuredrivecom.netlify.app
   - This enables: {company-slug}.epuredrive.com for each tenant

4. NETLIFY — Add wildcard domain
   - Go to: https://app.netlify.com/sites/epuredrivecom/domain-management
   - Add custom domain: *.epuredrive.com
   - Enable SSL for the wildcard (may require Netlify Pro plan)
   - Note: Wildcard SSL might need Netlify's paid plan

5. CSP HEADER — Update Content-Security-Policy
   - In netlify.toml, the img-src directive needs to include
     the tenant-logos bucket URL from Supabase storage
   - Add to img-src: https://brwzjwbpguiignrxvjdc.supabase.co/storage/v1/object/public/tenant-logos/
